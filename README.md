# Return-Class-Replacement-for-Bindable-Functions-

Return is a module that utilizes the OOP paradigm and replaces the default Bindable function Instance. Since it is made using pure lua it is inherently faster and will not have any memory leaks. Inspired by GoodSignal feel free to use this wherever you want!

## Getting Started

This project is built with [Rojo](https://github.com/rojo-rbx/rojo) 7.7.0-rc.1.

To build the place from scratch, use:

```bash
rojo build -o "Return Class Repo.rbxlx"
```

Next, open `Return Class Repo.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
